<title>Plant trees</title> <h1><strong>Plant trees</strong></h1> <p> The dialogue is divided into four sections. <br>  top left is a selection list of available tree types<br>  bottom left is shown a picture of the chosen tree<br>  top right some options may be set<br>  bottom right is information about the chosen tree given<br> </p> <h1>Selection list</h1> <p> In the selection list are given all  tree types that are available with the options specified top right. There are two tabs that will allow you to change how the tree types are identified.<br>  <em>Translation:</em> Shows the name of the tree in the given language. If there is no translation available in the given language, the object name will be used.<br>  <em>Object:</em> Shows the Simutrans internal object name for the tree type.<br> <br> </p> <h1>Options</h1> <p>  <em>ignore climates:</em> This option deactivates the climate restrictions associated with various tree types.<br>  <em>random year:</em> The age of the tree to be planted will be randomized.<br> </p>

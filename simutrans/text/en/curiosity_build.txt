<title>Build attraction buildings </title> 
<h1><strong>Build attraction buildings</strong></h1> 
<p>The dialogue is divided into four sections. 
<br>
  top left is a selection list of available buildings<br>
  bottom left is shown a picture of the chosen building<br>
  top right some options may be set<br>
  bottom right is information about the chosen city building given<br></p> 
<h1>Selection list</h1>
<p> 
In the selection list are given all special buildings and landmarks that are available with the options specified top right (<em>City attractions</em>, <em>Land attractions</em>, <em>Monuments</em>). 
There are two tabs that will allow you to change how the attraction buildings are identified.<br>
  <em>Translation:</em> Shows the name of the building in the given language. If there is no translation available in the given language, the object name will be used.<br> 
 <em>Object:</em> Shows the Simutrans internal object name for the building type.
<br> 
<br> 
The color of the text has the following meaning:<br>
  <em>blue text</em> attraction has to be built in cities<br>
  <em>green text</em> attraction has to be built on the countryside<br>
  <em>black text</em> monuments: can only be built once<br> 
</p> 
<h1>Options</h1>
<p>
  <em>ignore climates:</em> This option deactivates the climate restrictions associated with various building types.<br>
  <em>Use timeline year:</em> The selection list will only contain buildings that are available the current Simutrans year.<br>
  <em>Show obsolete too:</em> Expand the selection list to also contain buildings that are obsolete the current Simutrans year.<br>
  <em>City attraction:</em> There will be city attractions in the selection list.<br>
  <em>Land attraction:</em> There will be land attractions in the selection list.<br>
  <em>Monuments:</em> There will be monuments  in the selection list. <br>
  <em>Rotation:</em> If the chosen building can be shown from different angles, then it is possible to chose a specific view of it here. The image bottom left will change to reflect this. If this option is set to <em>random</em> a random view of the building will be used.<br> 
</p>
